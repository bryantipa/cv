PROFIL

Salam kenal, nama saya Bryan Ongen Tipawael. Saya lahir dan besar di Indonesia, tepatnya di kota Ambon. Saat ini, saya berusia 25 tahun dan telah menyelesaikan studi di Universitas Nani Bili Nusantara dengan gelar sarjana di bidang keahlian Web Developer. Sejak kecil, saya sudah tertarik dengan teknologi dan komputer. Ketertarikan ini membawa saya untuk belajar lebih banyak tentang dunia digital dan saya mulai mempelajari bahasa pemrograman sejak duduk di bangku SMA. Ketika saya memulai studi di Universitas Nani Bili Nusantara, saya tahu bahwa saya ingin memfokuskan diri di bidang pengembangan website dan aplikasi.

Selama kuliah, saya belajar banyak tentang HTML, CSS, Javascript, dan bahasa pemrograman lainnya yang diperlukan untuk membuat website dan aplikasi yang interaktif dan menarik. Saya juga mengikuti beberapa kursus dan pelatihan online untuk meningkatkan keahlian saya. Setelah lulus, saya bergabung dengan sebuah perusahaan teknologi yang bergerak di bidang pengembangan website dan aplikasi. Saya bekerja sebagai Web Developer dan bertanggung jawab untuk membuat website dan aplikasi yang inovatif dan menarik bagi pengguna.

Selama bekerja, saya juga belajar tentang manajemen proyek dan bekerja sama dengan tim lain untuk menghasilkan solusi yang terbaik bagi klien kami. Saya senang dengan karir saya sebagai Web Developer karena saya bisa terus mengembangkan keterampilan saya dalam dunia teknologi yang selalu berkembang.

Saya juga senang bisa membuat sesuatu yang bermanfaat dan dapat diakses oleh banyak orang. Terima kasih telah mendengarkan perkenalan diri saya, jika ada yang ingin ditanyakan silahkan tanyakan saja!
